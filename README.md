# 🔐 CipherVault - Secure File Storage System

## Project Overview

CipherVault is a secure file storage application that allows users to upload files securely.
Uploaded files are encrypted before storage and decrypted only during authorized download.

The project demonstrates secure backend development combined with DevOps practices.

---

## Features

✅ User Registration  
✅ JWT Authentication  
✅ Secure Login  
✅ Protected File Upload API  
✅ File Encryption using Fernet  
✅ Encrypted File Storage  
✅ Secure File Download and Decryption  
✅ Docker Containerization
